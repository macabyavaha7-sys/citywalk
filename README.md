# L_One神奇角落在哪里

城市人文摄影网站主页雏形。

## 文件结构

- `index.html`: 首页结构，包含顶部导航、推荐内容、作品展示和联系方式。
- `styles.css`: 页面视觉、固定导航、响应式网格和占位封面样式。
- `app.js`: 首页数据与交互，包括推荐内容、作品列表、搜索、分类筛选和分页。

## 后续替换内容

单篇文章页面完成后，把 `app.js` 里的 `featuredPost`、`posts`、`socialProfiles` 替换为真实标题、封面、链接、发布日期和平台主页地址即可。

当前页面是纯静态文件，可以直接打开 `index.html` 预览，也可以托管到 GitHub Pages。
